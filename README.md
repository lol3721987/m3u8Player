# M3U8视频播放器助手 - 多源版本

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
![Version](https://img.shields.io/badge/version-1.0.1-brightgreen.svg)

一款功能强大的浏览器扩展脚本，专为喜欢在线观看视频的用户设计。它能够智能检测网页中的M3U8视频链接，并提供便捷的播放功能。

## 核心功能

### 🎬 多源视频搜索
- 集成17个以上主流视频资源站
- 支持单一来源搜索和聚合搜索
- 智能分页显示，快速找到想看的内容
- 实时搜索结果展示

### ▶️ 智能视频播放
- 基于HLS.js技术，播放流畅稳定
- 自动检测网页中的M3U8链接
- 一键播放，无需手动复制链接
- 支持连续剧选集播放
- 多个CDN备用，确保播放稳定性

### 🖱️ 便捷操作体验
- 浮动搜索面板，不干扰正常浏览
- 支持拖拽移动，自由调整位置
- 自动记忆上次搜索和偏好设置
- 复制视频链接功能，方便分享

## 安装方式

### 前置准备
1. 确保使用Chrome、Firefox、Edge等现代浏览器
2. 安装[Tampermonkey](https://www.tampermonkey.net/)浏览器扩展

### 安装步骤
1. 打开浏览器，访问Tampermonkey官网安装扩展
2. 点击下方链接直接安装脚本：
   ```
   https://github.com/lol3721987/m3u8Player/raw/main/m3u8Player-tampermonkey.user.js
   ```
   或者： [greasyfork](https://greasyfork.org/zh-CN/scripts/544089-m3u8player)
3. Tampermonkey会自动弹出安装提示，点击"安装"即可

## 使用指南

### 启动播放器
- **方法一**：点击浏览器右上角的Tampermonkey图标，选择"🎬 打开视频搜索"
- **方法二**：在任意网页右下角找到圆形播放器按钮并点击

### 搜索视频
1. 在搜索框中输入想要观看的视频名称
2. 选择搜索模式：
   - **聚合搜索**：同时搜索所有可用资源站
   - **单一来源**：只搜索指定的视频源
3. 点击搜索按钮，等待结果加载

### 播放视频
1. 在搜索结果中找到想看的视频
2. 点击"播放"按钮开始观看
3. 播放器支持：
   - 播放/暂停控制
   - 进度条拖拽
   - 音量调节
   - 全屏播放

### 管理收藏
- 可以保存常用的视频源偏好
- 搜索历史自动记录
- 播放历史可查看

## 常见问题

### Q: 安装后无法使用？
A: 请确认已正确安装Tampermonkey扩展，并启用了本脚本。

### Q: 搜索不到想看的内容？
A: 尝试切换到聚合搜索模式，或检查网络连接是否正常。

### Q: 视频播放卡顿？
A: 网络波动可能导致播放不流畅，建议稍后重试或检查网络状态。

### Q: 如何更新到最新版本？
A: 重新访问安装链接，Tampermonkey会自动检测并提示更新。

### Q: 支持哪些浏览器？
A: 支持Chrome、Firefox、Edge、Safari等主流浏览器的最新版本。
